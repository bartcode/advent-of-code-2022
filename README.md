# Advent of Code 2022

Solutions in Python for the advent of code of 2022.

## Usage

```commandline
pip install .
aoc --help
```

```text
Usage: aoc [OPTIONS] COMMAND [ARGS]...

  Program to solve Advent of Code 2022 challenges.

Options:
  --help  Show this message and exit.

Commands:
  day01  Solve day 1.
  ...    Solve day n.

```
